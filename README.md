# Decision Journal

A desktop application for tracking decisions, rules, and compliance.

## Electron Version (New)

The application has been converted to a modern desktop app using Electron.

### Prerequisites

- Node.js (v20+ recommended)

### Setup and Running

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the application:
   ```bash
   npm start
   ```

### Features

- **Editable Table**: Click and edit cells directly.
- **Auto-save**: Automatically saves to CSV on edit.
- **Stats**: Real-time summary of compliance.
- **Chart**: Interactive trend chart.
- **Premium UI**: Modern aesthetic with glassmorphism touches.
